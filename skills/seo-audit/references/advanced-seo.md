# Advanced SEO Techniques

## Schema Markup

Use JSON-LD structured data for:
- **Article**: Blog posts with author, date, publisher
- **Product**: Products with price, availability, reviews
- **Course**: Courses with provider, description, offers
- **FAQ**: FAQ sections for rich snippet potential
- **HowTo**: Step-by-step guides
- **BreadcrumbList**: Navigation breadcrumbs

## Core Web Vitals

Key metrics to optimize:
- **LCP (Largest Contentful Paint)**: < 2.5s — optimize hero images, preload fonts
- **FID (First Input Delay)**: < 100ms — minimize JS execution
- **CLS (Cumulative Layout Shift)**: < 0.1 — set image dimensions, avoid dynamic content injection

## Internal Linking Strategy

- Create topic clusters with pillar pages and supporting content
- Use descriptive anchor text (not "click here")
- Link from high-authority pages to new content
- Maintain a flat site architecture (3 clicks from home to any page)
- Update old content with links to new relevant pages

## Keyword Research Framework

1. Seed keywords from business context and industry
2. Expand with question-based queries ("how to", "what is", "best")
3. Analyze search intent: informational, navigational, commercial, transactional
4. Prioritize by: relevance to business, search volume, competition level
5. Map keywords to content types: blog posts, landing pages, product pages
