## SEO Audit Playbook

When performing an SEO audit or optimizing content for search, follow this framework:

### 1. Technical Foundations
- Check page titles (50-60 chars) and meta descriptions (150-160 chars)
- Verify H1/H2/H3 heading hierarchy
- Ensure images have descriptive alt text
- Check for proper canonical URLs
- Validate Open Graph and Twitter Card meta tags

### 2. Content Quality
- Target one primary keyword per page
- Include the primary keyword in: title, H1, first paragraph, URL slug
- Use related keywords naturally throughout
- Aim for comprehensive coverage (1500+ words for pillar content)
- Include internal links to related content (3-5 per page)

### 3. Blog Post Optimization
- Structure with clear H2 sections for featured snippet potential
- Add a compelling meta description with a call to action
- Include schema markup for article type
- Optimize images: compress, descriptive filenames, alt text
- Add related posts section for internal linking

### 4. Landing Page SEO
- Focus on conversion-intent keywords
- Keep above-fold content keyword-rich
- Use testimonials and social proof (schema markup for reviews)
- Ensure fast page load (minimize JS, optimize images)

### 5. Reporting
When asked to audit, structure your response as:
- **Score**: Quick assessment (Good/Needs Work/Critical)
- **Findings**: Specific issues found
- **Recommendations**: Prioritized action items
- **Quick Wins**: Changes that can be made immediately
